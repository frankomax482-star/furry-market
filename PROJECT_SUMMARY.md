## 🚀 Как запустить

### 1. Быстрый старт (автоматический)

**Windows:**
```bash
start.bat
```

**Mac/Linux:**
```bash
chmod +x start.sh
./start.sh
```

### 2. Ручной запуск

**Backend (Terminal 1):**
```bash
cd backend
npm install
npm run dev
```

**Frontend (Terminal 2):**
```bash
cd frontend
npm install
npm run dev
```

### 3. Откройте браузер
- Frontend: `http://localhost:3000`
- API: `http://localhost:5000/api`
